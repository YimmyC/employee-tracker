# Employee Tracker

## About

Command line application designed to update, view, and add to an employee database.

## Features

- View all departments

- View all roles

- View all employees

- Add a department

- Add a role

- Add an employee

- Update employee

## How to use

Run node index to start. Choose from list of options.
